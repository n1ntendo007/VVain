# VVain

Мобильное PWA-приложение для официантов. Схема столиков фиксирована: столики нельзя добавлять, удалять или перемещать. Внутри каждого столика можно добавлять блюда свободным текстом, удалять отдельные строки и обнулять весь список.

## Хранение данных

Заказы хранятся только в `localStorage` конкретного браузера на конкретном устройстве. Сервер и база данных не используются. Между телефонами данные не синхронизируются.

## Публикация на Vercel

1. Распакуйте архив.
2. Загрузите содержимое папки в корень репозитория GitHub.
3. В Vercel выберите **Add New → Project**, импортируйте репозиторий и нажмите **Deploy**.
4. Build Command и Output Directory указывать не нужно: проект статический.

## Установка на телефон

### iPhone

Откройте опубликованный сайт в Safari → нажмите кнопку «Поделиться» → выберите «На экран Домой».

### Android

Откройте опубликованный сайт в Chrome → откройте меню браузера → выберите «Установить приложение» или «Добавить на главный экран».

## Файлы

- `index.html` — разметка двух экранов приложения.
- `styles.css` — мобильный интерфейс в красно-белом стиле.
- `app.js` — работа со столиками и локальное сохранение заказов.
- `manifest.webmanifest` и `service-worker.js` — установка PWA и офлайн-запуск.
- `vercel.json` — корректные заголовки для PWA на Vercel.

## Статичный масштаб на телефонах

Масштабирование страницы отключено. Поле ввода использует размер текста 16 px, поэтому iPhone не приближает экран при вводе блюда.
