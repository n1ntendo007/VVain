# VVain

Готовый PWA-проект для официантов.

## Что есть
- схема из 17 фиксированных столиков;
- зелёный цвет для пустых столиков;
- мягкий жёлтый цвет для столиков с заказами;
- количество позиций и время последнего изменения на активных столиках;
- отдельный экран столика;
- добавление блюд свободным текстом;
- удаление отдельных блюд;
- чаевые по каждому столику;
- общий итог чаевых за сегодня на главном экране;
- кнопка очистки всех чаевых;
- сохранение данных в localStorage;
- установка на экран телефона как PWA.

## Запуск
Просто загрузите проект в GitHub и импортируйте в Vercel.
