# VVain v8

Пересобранная версия приложения.

## Исправлено
- окно чаевых не существует в DOM до нажатия на кнопку «Чаевые столику»;
- окно чаевых закрывается по крестику, нажатию вне окна и Escape;
- у свободных столиков не отображается бейдж количества и время;
- у занятых столиков показываются количество позиций и последнее время;
- добавление и удаление блюд;
- чаевые по столику и общий итог;
- очистка всех чаевых;
- локальное хранение данных;
- PWA и защита от старого кеша благодаря versioned CSS/JS.

## Публикация
Распакуйте архив и загрузите файлы из корня архива в репозиторий. Затем импортируйте репозиторий в Vercel.
