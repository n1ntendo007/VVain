# VVain

Готовый статический PWA-проект для Vercel.

## Что есть
- 17 фиксированных столиков как на макете;
- пустой столик — зелёный;
- столик с заказами — жёлтый;
- на активных столиках показываются время последнего изменения и количество позиций;
- отдельный экран столика;
- свободный ввод блюд;
- удаление любой строки заказа;
- чаевые по каждому столику;
- общий итог чаевых на главном экране;
- кнопка очистки всех чаевых;
- локальное сохранение в браузере;
- установка на главный экран iPhone/Android.

## Публикация
Просто загрузи всё содержимое папки в репозиторий GitHub и импортируй проект в Vercel.
